# ASP.NET MVC 5
部落格文章的範例

## 需求
- .Net Framework 4.6
- [Northwind Database](https://www.microsoft.com/en-us/download/details.aspx?id=23654)

## 說明
利用 git 切換分支，每個分支會將功能範例盡量單純， develop 或是 master 則是功能統合。
### 分支說明
- ModelExt
說明 **edmx** 產生之對應類別，可以使用擴充方法來擴充，詳情可以參考[此篇](https://shunnien.github.io/2017/07/31/aspnet-mvc-model-extension-method/#more)
- modelMetadata
範例說明 **MetaData** 的使用，詳情可以參考[此篇](https://shunnien.github.io/2017/07/31/net-mvc-5-metadata/#more)

### 資料夾說明
- Identity 資料夾

## 使用方式
切換想要觀看的分支
``` cmd
git checkout 分支名稱
```
